# Facial_Expression_Recognition

* To run this project:

`docker run --name [name] -v [your_path_local/expression_recog]:/app/Workspace -it fernandaszadr/facial-expression-recognition`


* Label used to represent classes:
  * 0 -> Angry
  * 1 -> Disgust
  * 2 -> Fear
  * 3 -> Happy
  * 4 -> Neutral
  * 5 -> Sad
  * 6 -> Surprise
